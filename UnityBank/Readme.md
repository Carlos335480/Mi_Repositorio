# UnityBank

## Descripción
UnityBank es una plataforma web de **banca en línea**, donde los usuarios pueden registrarse, iniciar sesión y realizar operaciones bancarias de manera segura. El sistema permite al administrador gestionar usuarios y transacciones.

---

## Tecnologías usadas
- **Front-end:** HTML, CSS, JavaScript  
- **Back-end:** PHP  
- **Base de datos:** MySQL  
- **Estilo:** Bootstrap  
- **Envió de correos:** PHPMailer (para notificaciones a los usuarios)
