# TwinDriveMotors

## Descripción
TwinDriveMotors es una plataforma web para la **compra y venta de autos**, donde los usuarios pueden explorar autos disponibles y realizar compras, mientras que **el administrador gestiona los autos disponibles en la plataforma**.

---

## Tecnologías usadas
- **Front-end:** HTML, CSS, JavaScript  
- **Back-end:** PHP  
- **Base de datos:** MySQL  
- **Estilo:** Bootstrap
